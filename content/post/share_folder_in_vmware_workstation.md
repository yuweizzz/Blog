首先在宿主机创建好需要的共享文件夹，然后启动 VMware Workstation 并选择对应的虚拟机，依次点击`编辑虚拟机设置`，`选项`，`共享文件夹`，选择创建好的主机目录进行添加，宿主机需要做的部分就完成了，可以根据需要设置读写模式为只读，这样虚拟机就只能读取而不能修改共享目录。

接下来是虚拟机需要操作的部分：

``` bash
# 虚拟机需要安装 open-vm-tools
$ yum install open-vm-tools

# 安装完成后可以列出宿主机提供的共享目录
$ vmware-hgfsclient
Share  # 在宿主机指定共享目录的时候，还需要自定义名称，会在这里显示

# 挂载目录
$ mkdir /mnt/share
$ vmhgfs-fuse .host:/Share /mnt/share

# 开机自动挂载
$ echo .host:/Share /mnt/share fuse.vmhgfs-fuse allow_other 0 0 >> /etc/fstab 
```
